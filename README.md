# country-stats-cli

A Java command-line tool that filters and displays country statistics (population, area, and density) based on custom thresholds.

## Features

* Filters a dataset of over 200 countries.
* Outputs a cleanly formatted table of Population, Area, and Population Density.
* Allows ignoring specific filters using the `I` flag.

## Compilation

Ensure you have the Java Development Kit (JDK) installed. Navigate to the root directory of your project (the folder containing the `Program1` directory) and run:

```bash
javac Program1/CountryStats.java
